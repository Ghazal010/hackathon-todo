# Color Scheme - Phase 2

## Brand Colors

### Primary Palette
- **Rose Mist**: #EAC8CA
  - Usage: Accent colors, decorative elements, hover states
  - Accessibility: Good contrast with dark text

- **Lavender Dream**: #F2D5F8
  - Usage: Primary backgrounds, tags, light UI elements
  - Accessibility: Soft background color with good readability

- **Orchid Whisper**: #E6C0E9
  - Usage: Gradients, highlights, active states
  - Accessibility: Good for interactive elements

- **Purple Haze**: #BFABCB
  - Usage: Secondary elements, borders, secondary text
  - Accessibility: Medium contrast element

- **Slate Purple**: #8D89A6
  - Usage: Main content text, icons, primary content
  - Accessibility: Primary text color with good contrast

## Usage Guidelines

### Backgrounds
- Primary: Gradient from Lavender Dream to Orchid Whisper
- Secondary: Rose Mist with transparency
- Glass effect: rgba(255, 255, 255, 0.7) with backdrop-filter

### Text Colors
- Primary text: Slate Purple (#8D89A6)
- Secondary text: Purple Haze (#BFABCB)
- Light text: Orchid Whisper (#E6C0E9)

### Interactive Elements
- Buttons: Gradient from Orchid Whisper to Purple Haze
- Hover states: Rose Mist (#EAC8CA)
- Active states: Orchid Whisper (#E6C0E9)

### Task Priority Colors
- High Priority: #ff6b6b (red accent)
- Medium Priority: #feca57 (yellow accent)
- Low Priority: #48dbfb (blue accent)

### Dark Mode Adaptation
- Primary backgrounds: #1a1625 to #2a2435 gradient
- Glass effect: rgba(141, 137, 166, 0.2)
- Borders: rgba(191, 171, 203, 0.3)

## Accessibility Compliance
- All text meets WCAG AA contrast ratios
- Color is not the sole indicator of meaning
- Sufficient contrast for readability
